# BReg

## Overview

BReg is a course registration client bundled with a timetable planner.

## Features

-   Executing registration actions
-   Caching system
-   Timetable planner with preferences
-   Scriptable

## Installation

To install BReg, clone the repository and then install the required dependencies by running:

```bash
pip install -r requirements.txt
```

## Usage

To run BReg, execute the bootstrap script:

```bash
run.sh
```

or on Windows:

```bash
run.bat
```

For detailed usage instructions, please refer to the [MANUAL](docs/MANUAL.md) file.
